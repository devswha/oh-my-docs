## 什么是技能？

技能是以 `/oh-my-claudecode:skill-name` 格式调用的命令。
它们也可以通过魔法关键词激活，你可以在对话中使用它们而不需要记住命令名称。

> **数量说明：** OMC 内置 **37 个技能**。其中 36 个可通过斜杠命令或魔法关键词用户调用；1 个（`omc-reference`）自动加载内部使用且 `user-invocable: false`。

## 如何调用技能

### 斜杠命令

```bash
/oh-my-claudecode:autopilot "书店的 REST API"
/oh-my-claudecode:ralph "修复所有 TypeScript 错误"
/oh-my-claudecode:team 3:executor "实现用户认证"
```

### 魔法关键词

在对话中输入的关键词会自动激活技能。

```
autopilot build me a todo app with React
ralph fix the authentication module
ulw implement user management
```

## 技能关系图

```
deep-interview（澄清需求）
  → ralplan（共识规划）
    → autopilot（自主执行）
      → ralph（循环执行）
        → ultrawork（并行执行）
      → ultraqa（QA 循环）
```

更高层级的技能会自动包含较低层级的技能。
运行 autopilot 会在内部借用 ralph 和 ultrawork 的帮助。
