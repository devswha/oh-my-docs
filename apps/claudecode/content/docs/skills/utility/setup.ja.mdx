## 概要

Setup は OMC の設定関連スキルを 1 つのエントリポイントに統合するスキル。提供された引数に基づいて、適切なサブスキルにルーティングします。

## 使用方法

```bash
/oh-my-claudecode:setup              # フルセットアップウィザード（→ omc-setup）
/oh-my-claudecode:setup doctor       # インストール診断（→ omc-doctor）
/oh-my-claudecode:setup mcp          # MCP サーバー設定（→ mcp-setup）
/oh-my-claudecode:setup wizard       # 明示的なウィザードパス（→ omc-setup）
/oh-my-claudecode:setup --local      # ローカルプロジェクト設定（→ omc-setup --local）
/oh-my-claudecode:setup --global     # グローバル設定（→ omc-setup --global）
```

## ルーティングルール

| 引数 | ルーティング先 |
|----------|-----------|
| none、`wizard`、`local`、`global`、`--force` | `/oh-my-claudecode:omc-setup` |
| `doctor` | `/oh-my-claudecode:omc-doctor` |
| `mcp` | `/oh-my-claudecode:mcp-setup` |

## 互換性

`omc-setup`、`omc-doctor`、`mcp-setup` は引き続き個別に呼び出すことができます。
新しいドキュメントは `/oh-my-claudecode:setup` の使用を推奨します。
