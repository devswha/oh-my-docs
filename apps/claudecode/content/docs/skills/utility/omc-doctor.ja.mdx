## 概要

OMC ドクターは oh-my-claudecode のインストール状態を診断し、検出された問題を自動的に修正するスキルです。

## 使用方法

```bash
/oh-my-claudecode:omc-doctor
```

## 診断チェック

| チェック | ステータス | 説明 |
|---------|----------|------|
| プラグインバージョン | OK / WARN / CRITICAL | インストール済みかどうか、最新かどうか |
| レガシーフック（settings.json） | OK / CRITICAL | 重複実行を引き起こすレガシーフック |
| レガシースクリプト（~/.claude/hooks/） | OK / WARN | 混乱を引き起こすレガシースクリプト |
| CLAUDE.md | OK / WARN / CRITICAL | OMC 設定マーカーが存在するかどうか |
| プラグインキャッシュ | OK / WARN | キャッシュされた複数バージョンが存在するかどうか |
| レガシーエージェント/コマンド/スキル | OK / WARN | curl インストールからの残存成果物 |

## 診断レポート

```
## OMC ドクター レポート

### サマリー
[健全 / 問題が見つかりました]

### チェック
| チェック              | ステータス   | 詳細 |
|--------------------|----------|---------|
| プラグインバージョン | OK       | 4.11.4  |
| レガシーフック       | CRITICAL | ...     |
| CLAUDE.md          | OK       | ...     |
```

## 自動修復

問題が見つかった場合、自動修復が提案されます。

| 問題 | 修復 |
|-----|------|
| レガシーフック | settings.json から `hooks` セクションを削除 |
| レガシースクリプト | `~/.claude/hooks/*.sh` を削除 |
| 古いプラグイン | プラグインキャッシュをクリーンアップ |
| キャッシュ重複 | 最新バージョンのみを保持 |
| 欠落/古い CLAUDE.md | 最新 CLAUDE.md をダウンロード |
| レガシーインストール成果物 | バックアップして削除 |

修復後、Claude Code を再起動する必要があります。
