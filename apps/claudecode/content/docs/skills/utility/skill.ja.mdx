## 概要

Skill は oh-my-claudecode スキルを管理するメタスキル。ユーザーレベルおよびプロジェクトレベルの両方でローカルスキルをリスト、追加、削除、検索、編集、同期できます。

## 使用方法

```bash
/oh-my-claudecode:skill list
/oh-my-claudecode:skill add [name]
/oh-my-claudecode:skill remove <name>
/oh-my-claudecode:skill edit <name>
/oh-my-claudecode:skill search <query>
/oh-my-claudecode:skill info <name>
/oh-my-claudecode:skill sync
/oh-my-claudecode:skill setup
/oh-my-claudecode:skill scan
```

## サブコマンド

### list

ユーザースキル（`~/.claude/skills/omc-learned/`）とプロジェクトスキル（`.omc/skills/`）をスコープでまとめて表示します。

### add

インタラクティブウィザード経由で新しいスキルを作成。名前、説明、トリガーキーワード、スコープ（ユーザー/プロジェクト）を尋ね、その後 SKILL.md テンプレートを生成します。

### remove

名前でスキルを見つけ、確認後に削除します。確認なしで削除することはありません。

### edit

既存のスキルの説明、トリガー、引数ヒント、コンテンツをインタラクティブに変更します。

### search

クエリを名前、説明、トリガー、フルマークダウンコンテンツ全体で検索。名前/トリガーマッチはコンテンツマッチより優先度が高い。

### info

スキルの詳細情報とフルコンテンツを表示。

### sync

ユーザーとプロジェクトスコープ間のスキルを比較し、それらの間でコピーできるようにします。上書きする前に常に確認。

### setup

フルセットアップウィザードを実行。ディレクトリ検証、スキルスキャン、新しいスキル追加、会話パターンスキャン、スキルインポートを提供します。

## スキルストレージロケーション

| スコープ | パス | 目的 |
|-------|------|---------|
| ユーザー | `~/.claude/skills/omc-learned/` | すべてのプロジェクト全体で利用可能 |
| プロジェクト | `.omc/skills/` | このリポジトリでのみ使用（バージョン管理） |
