## スキルとは

スキルは `/oh-my-claudecode:スキル名` という形式で呼び出されるコマンドです。
マジックキーワードで有効化することもできるため、コマンド名を暗記せずに会話の中で使用できます。

> **数について:** OMC は **37 個の組み込みスキル** を搭載しています。36 個はスラッシュコマンドまたはマジックキーワードでユーザーが呼び出せます。1 個（`omc-reference`）は内部で自動ロードされ、`user-invocable: false` に設定されています。

## スキルの呼び出し方

### スラッシュコマンド

```bash
/oh-my-claudecode:autopilot "REST API for a bookstore"
/oh-my-claudecode:ralph "fix all TypeScript errors"
/oh-my-claudecode:team 3:executor "implement user auth"
```

### マジックキーワード

会話中にキーワードを入力するとスキルが自動的に有効化されます。

```
autopilot build me a todo app with React
ralph fix the authentication module
ulw implement user management
```

## スキル関係図

```
deep-interview (clarify requirements)
  → ralplan (consensus planning)
    → autopilot (autonomous execution)
      → ralph (loop execution)
        → ultrawork (parallel execution)
      → ultraqa (QA cycling)
```

より高度なスキルは低度なスキルを自動的に含みます。
autopilot を実行すると内部的に ralph と ultrawork の支援を求めます。
