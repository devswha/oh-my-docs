---
title: エンジニアリングスキル
description: Browser、UI、git スキルはワーカーに専門性を追加します。
---

| Skill | Use |
|---|---|
| `frontend-ui-ux` | Design-first UI implementation. |
| `playwright` | Browser automation through MCP. |
| `playwright-cli` | Browser automation through shell commands. |
| `agent-browser` | Navigation, snapshots, clicks, and forms. |
| `dev-browser` | Persistent browser page state. |
| `git-master` | Atomic commits, rebases, and history search. |
