LazyCodex は OmO plugin の Skills を Codex にインストールします。Commands (`$ulw-plan`、`$start-work`、`$ulw-loop`、`$init-deep`) は workflow を作り、Skills はその workflow 内で必要な専門判断を提供します。

Skills は Codex session 内で load されます。ユーザーが明示的に「この Skill を使って」と言うこともでき、task 内容が Skill description と合う場合は Codex がその playbook に従います。

## Documented Skills

| Skill | 用途 |
| --- | --- |
| [`review-work`](/ja/docs/skills/review-work) | 多角度の実装後レビュー |
| [`remove-ai-slops`](/ja/docs/skills/remove-ai-slops) | 挙動を保ったまま AI っぽいコードを整理 |
| [`debugging`](/ja/docs/skills/debugging) | 体系的な根本原因デバッグ |
| [`frontend`](/ja/docs/skills/frontend-ui-ux) | generic layout ではなく design された UI work |
| [`programming`](/ja/docs/skills/programming) | 厳格な TypeScript、Rust、Python、Go discipline |
| [`refactor`](/ja/docs/skills/refactor) | AST-grep による structural search/rewrite を含む、test-verified refactoring |
| [`LSP`](/ja/docs/skills/lsp) | diagnostics、definitions、references、symbols、rename |
| [`rules`](/ja/docs/skills/rules) | AGENTS、rules、instruction files からの project instructions |
| [`comment-checker`](/ja/docs/skills/comment-checker) | edit-like work 後の feedback |

## Other Installed Skills

`4.12.1` 時点の plugin には以下の Skills も含まれます。この docs site ではまだ個別 page を作っていませんが、Codex session では Skill として load できます。

| Skill | 用途 |
| --- | --- |
| `ast-grep` | AST-aware structural search と rewrite |
| `git-master` | commits、history investigation、rebase/squash/fixup などの git work |
| `lsp-setup` | language server install と configuration |
| `lcx-doctor` | LazyCodex/Codex install health diagnosis |
| `lcx-report-bug` | LazyCodex/Codex bug issue drafting |
| `lcx-contribute-bug-fix` | verified bug-fix contribution workflow |
| `teammode` | 複数 Codex threads を team のように coordinating |
| `ultraresearch` | 明示的に要求された deep research |
| `visual-qa` | UI/TUI 変更後の screenshot と visual verification |

Commands 自体も Skills として install されますが、ユーザー向け docs では [Commands](/ja/docs/commands) に分けています。

## Where Skills Fit

- Commands が flow を持ちます: planning、execution、verified completion。
- Skills はその flow の周辺に専門判断を足します: review、cleanup、UI work、language discipline、code intelligence。
- Skills は該当作業でだけ load され、終わると降りるため、agent prompt が軽く保たれます。
