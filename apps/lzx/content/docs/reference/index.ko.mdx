이 섹션은 LazyCodex(LZX) 런타임 Reference입니다. LazyCodex는 [OmO(oh-my-openagent)](https://github.com/code-yeongyu/oh-my-openagent)를 Codex용으로 패키징한 얇은 배포판입니다. LazyCodex로 실행하는 깊은 워크플로우는 OmO 플러그인의 Skills, hooks, MCP Servers, model profiles 위에서 동작합니다.

## What LazyCodex Owns

LazyCodex 자체는 작습니다. 제공하는 것은 하나입니다 — `npx`로 호출하며 Codex 플랫폼을 대상으로 OmO를 설치하고 구동하는 `lazycodex-ai` npm bin입니다.

```bash
npx lazycodex-ai install
```

이는 `npx --yes --package oh-my-openagent omo install --platform=codex`의 단축형입니다. 전역 설치는 없습니다 — 항상 `npx`로 호출합니다. Codex light 설정은 나중에 덮어쓸 수 있는 zero-config 기본값과 함께 제공됩니다.

## What OmO Owns

설치가 끝나면 그 아래의 OmO 기반 설정이 나머지를 담당합니다 — Agent Roles, 병렬 탐색, 모델 라우팅, Skills, hooks, 검증된 완료입니다. 설치 플래그를 넘어 provider·모델·동작을 조정하려면, 사실상 OmO를 설정하는 것입니다.
