このセクションは LazyCodex (LZX) runtime Reference です。LazyCodex は [OmO (oh-my-openagent)](https://github.com/code-yeongyu/oh-my-openagent) を Codex 向けに packaging する薄い配布です。LazyCodex で実行する深い workflows は、OmO plugin の Skills、hooks、MCP Servers、model profiles の上で動作します。

## What LazyCodex Owns

LazyCodex 自体は小さいものです。提供するのは 1 つだけです。`npx` で呼び出し、Codex platform 向けに OmO を install して動かす `lazycodex-ai` npm bin です。

```bash
npx lazycodex-ai install
```

これは `npx --yes --package oh-my-openagent omo install --platform=codex` の短縮形です。global install はありません。常に `npx` で呼び出します。Codex light setup は、あとから上書きできる zero-config defaults を備えています。

## What OmO Owns

install 後は、その下の OmO-based setup が残りを担当します: Agent Roles、parallel exploration、model routing、Skills、hooks、verified completion。install flags を超えて provider、model、behavior を調整する場合、実質的には OmO を設定しています。
