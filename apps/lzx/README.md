# lzx-docs

Official documentation site for [LazyCodex](https://github.com/code-yeongyu/lazycodex) (LZX), the [OmO (oh-my-openagent)](https://github.com/code-yeongyu/oh-my-openagent) agent harness packaged for Codex.

- Framework: Next.js 16 + Fumadocs 16
- Languages: English, 한국어, 中文, 日本語
- Source project: https://github.com/code-yeongyu/lazycodex

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run lint
npm run build
```
