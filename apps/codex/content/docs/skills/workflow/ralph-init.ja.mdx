## 一行要約

`$ralph-init` は `$ralph` ループを始める前に状態を初期化します。ralph が前のセッション状態を引き継いで誤動作するときに先に使います。

## 呼び出し

```
$ralph-init
```

## いつ使うか

- `$ralph` が前の作業状態を引き継いで誤動作するとき
- セッションを新たに始めて ralph ループをクリーンに初期化したいとき
- ralph ループが途中で止まったり状態が壊れた場合

## 動作方法

1. `.omx/state/` から ralph 関連の状態ファイルを整理します。
2. ループカウンターと完了証拠トラッキングを初期化します。
3. 初期化完了後に `$ralph` を実行する準備が整います。

## 通常の手順

```
$ralph-init
$ralph "承認された計画を最後まで完了して"
```
