## 一行要約

`$deepsearch` は単純な Grep 以上の深層探索が必要なときに使います。構造・パターン・依存関係を複数のエージェントが並列で分析し、結果をまとめます。

## 呼び出し

```
$deepsearch "認証ミドルウェアがどこでどのように使われているか全て把握して"
```

## いつ使うか

| 合う | 避ける |
|---|---|
| 変更前に影響範囲を把握する必要があるとき | ファイル一つ・関数一つ単位の探索 |
| レガシーコードからパターンを見つける必要があるとき | すでに範囲が明確な作業 |
| 複数モジュールにまたがる依存関係分析 | |

## 動作方法

1. explore エージェントがファイル・シンボルマップを作ります。
2. 関連領域を並列で探索します。
3. 結果をまとめてサマリーとファイル一覧を出力します。

結果は `$ralplan` や `$ralph` にコンテキストとしてそのまま続けて渡せます。
