## 한 줄 요약

`$frontend-ui-ux`는 화면 설계, 컴포넌트 구조, 인터랙션 흐름처럼 프론트엔드 특화 작업을 다룹니다. designer 에이전트를 중심으로 실행됩니다.

## 호출

```
$frontend-ui-ux "대시보드 레이아웃과 필터 컴포넌트를 설계해줘"
```

## 언제 쓰나

| 잘 맞는 경우 | 피할 경우 |
|---|---|
| UI 구조·인터랙션 설계 | 백엔드 로직만 다루는 작업 |
| 컴포넌트 계층과 상태 흐름 정의 | 데이터 모델·API 설계 |
| 접근성·반응형 레이아웃 요구 | |

## 동작 방식

1. 요구사항에서 화면 단위와 컴포넌트를 도출합니다.
2. 인터랙션 흐름과 상태 관리 방향을 정합니다.
3. 코드 또는 설계 문서로 결과를 출력합니다.

## `$team`과 함께 쓰기

프론트엔드·백엔드가 병렬로 진행 가능하다면 `$team`과 조합할 수 있습니다.

```
$team 2:executor "frontend-ui-ux 스펙대로 컴포넌트 구현, 백엔드 API 병렬 작성"
```
