## 한 줄 요약

`$deepsearch`는 단순 Grep 이상의 심층 탐색이 필요할 때 씁니다. 구조·패턴·의존성을 여러 에이전트가 병렬로 분석해서 결과를 모읍니다.

## 호출

```
$deepsearch "인증 미들웨어가 어디서 어떻게 쓰이는지 전부 파악해줘"
```

## 언제 쓰나

| 잘 맞는 경우 | 피할 경우 |
|---|---|
| 변경 전 영향 범위를 파악해야 할 때 | 파일 하나·함수 하나 단위 탐색 |
| 레거시 코드에서 패턴을 찾아야 할 때 | 이미 범위가 명확한 작업 |
| 여러 모듈에 걸친 의존성 분석 | |

## 동작 방식

1. explore 에이전트가 파일·심볼 지도를 만듭니다.
2. 관련 영역을 병렬로 탐색합니다.
3. 결과를 종합해서 요약과 파일 목록을 출력합니다.

결과물은 이후 `$ralplan`이나 `$ralph`에 컨텍스트로 바로 이어붙일 수 있습니다.

## 관련 스킬

- [`$ralplan`](/ko/docs/skills/workflow/ralplan) — 탐색 결과를 계획으로 이어가기
- [`$ralph`](/ko/docs/skills/workflow/ralph) — 파악한 범위로 실행
