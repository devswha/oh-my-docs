## ユーティリティスキルとは

ユーティリティスキルは、アクティブな OMX ワークフローを補助しますが、主実行モードそのものにはなりません。状態確認、別モデルへの短い相談、インストール問題の診断、作業メモの保存、集中レビューを行うときに使います。
