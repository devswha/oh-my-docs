## Overview

`$help` prints the skills, agents, and key commands available in the current OMX session. Use it when you don't know what skills exist or want to quickly check how a specific skill is used.

## Example

```text
$help
$help ralph
$help team
```

Running without arguments prints the full skill list. Providing a skill name prints that skill's description and usage.

## When to use

- When first starting with OMX and you want to see what skills are available
- When you've forgotten the exact invocation for a specific skill
- When you want to check the list of currently installed skills

## Note

The full skill reference is also available in the [skills documentation](/docs/skills).
