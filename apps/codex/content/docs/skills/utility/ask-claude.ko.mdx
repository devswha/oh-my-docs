`$ask-claude`로 호출합니다.

현재 Codex 세션을 벗어나지 않고 Claude에게 서브 질문을 던지는 브리지입니다. 위임된 호출은 별도 Claude 컨텍스트에서 실행되므로, 현재 작업 흐름과 분리된 답을 얻을 수 있습니다.

## 언제 쓰나

- Codex가 답변하기 어려운 추론 집약적 질문을 Claude에게 맡기고 싶을 때
- 현재 컨텍스트와 독립적인 요약이나 분석이 필요할 때
- Gemini와 나란히 다른 모델의 시각을 비교할 때
- 짧고 범위가 명확한 서브 질문을 빠르게 처리할 때

## 언제 피하나

- 현재 세션의 코드베이스 맥락이 필요한 질문 → Codex에서 직접 처리
- 복잡한 구현 작업 → `$ralph` 또는 executor 에이전트

## 명령

```bash
$ask-claude "멱등 HTTP 엔드포인트의 가장 안전한 retry 전략이 뭐야?"
$ask-claude --print "이 로그를 다섯 줄로 요약해줘"
$ask-claude --agent-prompt writer "이 메모를 릴리스 노트로 바꿔줘"
```

## 동작 방식

새 Claude 컨텍스트를 열어 질문을 전달하고 결과를 현재 세션으로 돌려줍니다. 현재 작업 세션의 파일이나 상태는 물려받지 않습니다.

기본 워크플로우처럼 자주 쓰기보다는, Claude의 관점이 실제로 필요한 한정된 문제에서 의도적으로 사용하는 쪽이 좋습니다.
