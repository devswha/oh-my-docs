`$ask-gemini`로 호출합니다.

현재 Codex 세션을 벗어나지 않고 Gemini에게 서브 질문을 던지는 브리지입니다. `$ask-claude`와 구조는 같고, 대상 모델이 Gemini로 다릅니다.

## 언제 쓰나

- Gemini의 긴 컨텍스트 처리 능력이 필요한 질문이 있을 때
- Claude와 Gemini의 답변을 나란히 비교하고 싶을 때
- 현재 컨텍스트와 독립된 서브 질문을 빠르게 처리할 때

## 언제 피하나

- 현재 코드베이스 맥락이 필요한 질문 → Codex에서 직접 처리
- Claude와 Gemini를 같은 의사결정 루프에서 비교하려면 제한된 `$ask-claude`/`$ask-gemini` 호출을 나란히 쓰거나 명시적인 team orchestration으로 합성하세요

## 명령

```bash
$ask-gemini "이 설계의 잠재적 병목을 찾아줘"
$ask-gemini --print "이 에러 로그 분석해줘"
$ask-gemini --agent-prompt analyst "이 요구사항을 정리해줘"
```

## 동작 방식

별도 Gemini 컨텍스트를 열어 질문을 전달하고 결과를 현재 세션으로 돌려줍니다. 현재 세션의 파일이나 상태는 물려받지 않습니다.
