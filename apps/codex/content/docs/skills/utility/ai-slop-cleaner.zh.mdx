通过 `$ai-slop-cleaner` 调用。

这是一个去除AI生成文本中反复出现的套话的技能。用于整理文档、代码注释、PR说明等带有AI写作痕迹的文本，使其更自然。

## 何时使用

- 代码注释或文档充满了AI特有的文体（"通过这种方式，提供更加优化的体验…"）时
- PR说明、提交消息、README明显是AI写的时
- 想把OMX生成的文档草稿整理得像人写的一样时

## 何时避免

- 需要更改代码逻辑本身 → 使用 `$build-fix` 或直接修改
- 问题在于内容准确性而非文体 → 使用 `$analyze` 或 `$code-review`

## 命令

```bash
$ai-slop-cleaner "帮我整理一下这个PR说明"
$ai-slop-cleaner --file README.md
```

## 去除什么

| 模式 | 示例 |
|---|---|
| 夸张副词 | "非常高效地"、"更加优化的" |
| 重复总结 | 每段结尾加"即，…" |
| 不必要的开场白 | "本文档将介绍关于…的内容" |
| 无意义连接词 | "此外"、"同时"、"通过这种方式"的滥用 |
| 过度斜体/强调 | 每句话都加**粗体**处理 |
