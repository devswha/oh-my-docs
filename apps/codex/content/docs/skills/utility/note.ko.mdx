## 개요

`$note`는 현재 세션에 메모를 저장합니다. 나중에 참조할 결정 사항, 진행 상황, 임시 컨텍스트를 기록해 두는 데 씁니다. 저장된 메모는 같은 세션 내에서 에이전트들이 참조할 수 있습니다.

## 사용 예시

```text
$note "auth 리팩터는 JWT 방식으로 확정"
$note "현재 users 테이블 마이그레이션 보류 중"
$note
```

인자 없이 호출하면 저장된 메모 목록을 출력합니다.

## 언제 쓰는가

- 긴 세션에서 중간 결정 사항을 잃지 않으려 할 때
- `$ralph`나 `$team` 실행 중 에이전트 간 공유할 컨텍스트를 남길 때
- 나중에 돌아와서 확인할 임시 메모가 필요할 때

## 언제 피하는가

- 영구적으로 남겨야 하는 내용 — 세션 메모는 세션 범위입니다. 장기 기록은 `.omx/notepad.md`나 프로젝트 문서에 남기세요.
- 코드 변경 내용 — 메모가 아니라 실제 파일을 수정하세요.
