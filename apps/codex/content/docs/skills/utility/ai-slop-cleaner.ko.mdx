`$ai-slop-cleaner`로 호출합니다.

AI가 생성한 텍스트에서 반복되는 상투적 표현을 걷어내는 스킬입니다. 문서, 코드 주석, PR 설명 등 AI가 쓴 흔적이 남은 텍스트를 자연스럽게 다듬을 때 씁니다.

## 언제 쓰나

- 코드 주석이나 문서가 AI 특유의 문체("이를 통해 더욱 향상된 경험을...")로 가득할 때
- PR 설명, 커밋 메시지, README가 AI가 쓴 티가 역력할 때
- OMX가 생성한 문서 초안을 사람이 직접 쓴 것처럼 정리하고 싶을 때

## 언제 피하나

- 코드 로직 자체를 변경해야 하는 경우 → `$build-fix` 또는 직접 수정
- 문체가 아닌 내용의 정확성이 문제인 경우 → `$analyze` 또는 `$code-review`

## 명령

```bash
$ai-slop-cleaner "이 PR 설명 좀 다듬어줘"
$ai-slop-cleaner --file README.md
```

## 무엇을 제거하나

| 패턴 | 예시 |
|---|---|
| 과장된 부사 | "매우 효율적으로", "보다 향상된" |
| 반복 요약 | 문단마다 붙는 "즉, ...입니다" |
| 불필요한 서론 | "이 문서에서는 ...에 대해 알아봅니다" |
| 무의미한 연결어 | "또한", "더불어", "이를 통해" 남발 |
| 과도한 이탤릭/강조 | 모든 문장에 **굵게** 처리 |
