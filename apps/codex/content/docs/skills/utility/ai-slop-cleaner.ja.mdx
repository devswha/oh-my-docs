`$ai-slop-cleaner`で呼び出します。

AIが生成したテキストに繰り返し現れる常套句を取り除くスキルです。ドキュメント、コードコメント、PR説明などAIが書いた痕跡が残るテキストを自然に整えたいときに使います。

## いつ使うか

- コードコメントやドキュメントがAI特有の文体（「これにより、より向上した体験を…」）で溢れているとき
- PR説明、コミットメッセージ、READMEにAIが書いた雰囲気が強く出ているとき
- OMXが生成したドキュメント草稿を人が書いたように整理したいとき

## いつ避けるか

- コードロジック自体を変更しなければならない場合 → `$build-fix` または直接修正
- 文体ではなく内容の正確性が問題の場合 → `$analyze` または `$code-review`

## コマンド

```bash
$ai-slop-cleaner "このPR説明を整えてくれ"
$ai-slop-cleaner --file README.md
```

## 何を除去するか

| パターン | 例 |
|---|---|
| 誇張した副詞 | 「非常に効率的に」「より向上した」 |
| 繰り返し要約 | 段落ごとにつく「つまり、…です」 |
| 不要な前置き | 「このドキュメントでは…について説明します」 |
| 無意味な接続語 | 「また」「さらに」「これにより」の乱用 |
| 過剰なイタリック/強調 | すべての文に**太字**処理 |
