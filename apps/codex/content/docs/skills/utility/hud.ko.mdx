## 개요

`$hud`는 OMX 세션 상단에 표시되는 HUD(상태 표시줄)를 제어합니다. 현재 어떤 정보가 표시되는지 확인하거나, 표시 항목을 켜고 끄는 데 씁니다.

HUD는 활성 모드, 에이전트 상태, 현재 task 진행 상황 같은 세션 정보를 실시간으로 보여줍니다.

## 사용 예시

```text
$hud
$hud off
$hud on
$hud status
```

인자 없이 호출하면 현재 HUD 설정을 출력합니다.

## 언제 쓰는가

- 세션 상태가 HUD에 표시되는지 확인하고 싶을 때
- 긴 실행 중 HUD가 출력을 방해할 때 끄고 싶을 때
- `$ralph`나 `$team` 같은 장기 실행 모드에서 진행 상황을 추적할 때

## 언제 피하는가

- HUD와 무관한 작업 — `$hud`는 표시 설정만 다룹니다. 실행 모드나 에이전트를 제어하지 않습니다.
