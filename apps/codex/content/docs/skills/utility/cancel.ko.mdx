title: $cancel
description: 현재 실행 중인 OMX 모드를 종료하는 유틸리티 스킬
---

`$cancel`로 호출합니다.

`$ralph`, `$autopilot`, `$team` 등 활성 실행 모드를 중단합니다. 작업이 잘못된 방향으로 가거나 더 이상 진행이 필요 없을 때 씁니다.

## 언제 쓰나

- `$ralph`가 루프를 돌고 있는데 중단해야 할 때
- `$autopilot`이 기대와 다른 방향으로 진행될 때
- `$team`의 특정 lane을 멈춰야 할 때
- 작업이 완료됐거나 요구사항이 바뀌어 더 진행할 필요가 없을 때

## 언제 피하나

- 일시 중지가 아닌 완전히 다른 작업을 시작하려면 그냥 새 스킬을 호출하면 됨
- 작업이 아직 진행 중이면 취소하지 말고 완료될 때까지 기다리는 것을 권장

## 명령

```bash
$cancel
$cancel --force
```

`--force`는 모든 활성 상태를 초기화합니다. 일반적으로는 인자 없이 쓰면 됩니다.

## 동작 방식

현재 활성 실행 모드의 상태를 읽고 중단 신호를 보냅니다. 진행 중이던 에이전트 작업은 현재 단계를 마무리한 뒤 멈춥니다. 부분 완료된 파일은 그 상태로 남습니다.

`$cancel` 후 재개하려면 중단된 시점부터 스킬을 다시 호출하면 됩니다.
