## 유틸리티 스킬이란?

유틸리티 스킬은 활성 OMX 워크플로우를 보조하지만 주 실행 모드 자체가 되지는 않습니다. 상태를 확인하거나, 다른 모델에 짧게 묻거나, 설치 문제를 진단하거나, 작업 메모를 저장하거나, 집중 리뷰를 실행할 때 사용합니다.

## 관련 워크플로우 스킬

보조 도구가 아니라 주 실행 lane이 필요하다면 [스킬 개요](/ko/docs/skills)로 돌아가 워크플로우 스킬을 선택하세요.
