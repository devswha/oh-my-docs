`$cancel`で呼び出します。

`$ralph`、`$autopilot`、`$team` などアクティブな実行モードを中断します。作業が間違った方向に進んでいるか、これ以上進める必要がないときに使います。

## いつ使うか

- `$ralph` がループを回しているが中断しなければならないとき
- `$autopilot` が期待と異なる方向に進んでいるとき
- `$team` の特定のlaneを止めなければならないとき
- 作業が完了したか要件が変わってこれ以上進める必要がないとき

## いつ避けるか

- 一時停止ではなく完全に別の作業を始めたい場合は、そのまま新しいスキルを呼び出せばよい
- 作業がまだ進行中であれば取り消さず完了まで待つことを推奨

## コマンド

```bash
$cancel
$cancel --force
```

`--force` はすべてのアクティブな状態を初期化します。通常は引数なしで使えばよいです。

## 動作方式

現在のアクティブな実行モードの状態を読み取り、中断シグナルを送ります。進行中だったエージェント作業は現在のステップを終えてから止まります。部分的に完了したファイルはその状態のまま残ります。

`$cancel` 後に再開したいときは中断した時点からスキルを再度呼び出してください。
