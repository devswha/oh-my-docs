## 개요

`$help`는 현재 OMX 세션에서 쓸 수 있는 스킬, 에이전트, 주요 명령어를 출력합니다. 어떤 스킬이 있는지 모를 때나 특정 스킬의 사용법을 빠르게 확인할 때 씁니다.

## 사용 예시

```text
$help
$help ralph
$help team
```

인자 없이 호출하면 전체 스킬 목록을 출력합니다. 스킬 이름을 인자로 주면 해당 스킬의 설명과 사용법을 출력합니다.

## 언제 쓰는가

- OMX를 처음 쓰기 시작할 때 어떤 스킬이 있는지 파악
- 특정 스킬의 정확한 호출 방식을 잊었을 때
- 현재 설치된 스킬 목록을 확인할 때
