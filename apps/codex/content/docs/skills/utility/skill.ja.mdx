## 概要

`$skill` は現在の環境にインストールされているスキルを管理します。どんなスキルがあるか一覧を見たり、新しいスキルを追加したり、既存のスキルを削除するのに使います。

## 使用例

```text
$skill list
$skill add my-workflow
$skill remove my-workflow
```

| コマンド | 説明 |
|---|---|
| `$skill list` | インストール済みスキルの全一覧を出力 |
| `$skill add <name>` | 新しいスキルを追加 |
| `$skill remove <name>` | スキルを削除 |

## いつ使うか

- 現在使用可能なスキル一覧を確認するとき
- カスタムスキルを追加または削除するとき
- インストールされているスキルが正しく配置されているか点検するとき

## いつ避けるか

- スキルの内容を見たり使い方を確認するとき — `$help <skill-name>` を使ってください。
- 環境全体の診断 — `$doctor` を使ってください。
- スキルのインストールパス設定の変更 — `$omx-setup` を使ってください。
