**OpenAI Codex CLI のためのマルチエージェント・オーケストレーション。学習曲線ゼロ。**

_Codex のプロンプトを覚える必要はありません。OMX を使うだけです。_

Oh My CodeX (OMX) は OpenAI Codex CLI の上で動作するワークフローレイヤーです。Codex をコーディングエンジンとして使いながら、曖昧なタスクを要件整理、承認済み計画、実装、検証へ進めるドキュメント型の流れを追加します。

## クイックスタート

<Steps>
  <Step>
    ### Codex CLI をインストール

    ```bash
    npm i -g @openai/codex
    codex login
    ```
  </Step>
  <Step>
    ### OMX をインストールして設定

    ```bash
    npm i -g oh-my-codex
    omx setup
    ```
  </Step>
  <Step>
    ### インストールを検証

    ```bash
    omx doctor
    codex login status
    omx exec --skip-git-repo-check -C . "Reply with exactly OMX-EXEC-OK"
    ```
  </Step>
  <Step>
    ### 最初のタスクを送る

    ```bash
    omx --madmax --high
    ```

    セッション内で標準の guided flow を実行します。

    ```text
    $deep-interview "clarify the authentication change"
    $ralplan "approve the safest implementation path"
    $ultragoal "turn the approved plan into durable goals and complete them"
    ```
  </Step>
</Steps>
