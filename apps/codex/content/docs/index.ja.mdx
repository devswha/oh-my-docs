**OpenAI Codex CLI 向けマルチエージェントオーケストレーション。学習コストゼロ。**

_Codex プロンプトを学ぶ必要はありません。OMX をそのまま使ってください。_

Oh My CodeX (OMX) は OpenAI Codex CLI の上で動作するワークフローレイヤーです。1 回の呼び出しで専用エージェントプロンプトを連携させ、複雑な開発タスクをエンドツーエンドで自動化します。

## クイックスタート

### 第 1 ステップ — Codex CLI をインストール

```bash
npm i -g @openai/codex
```

### 第 2 ステップ — OMX をインストール

```bash
npm i -g oh-my-codex
```

### 第 3 ステップ — セットアップ

```bash
omx setup
```

### 第 4 ステップ — 最初のセッションのスモークテストを実行

```bash
omx doctor
codex login status
omx exec --skip-git-repo-check -C . "Reply with exactly OMX-EXEC-OK"
```

### 第 5 ステップ — 推奨方法で OMX を起動

```bash
omx --madmax --high
```

そのセッション内で、標準ワークフローを試してください:

```
$deep-interview "認証変更の要件を明確にして"
$ralplan "もっとも安全な実装パスを承認して"
$ralph "承認済みの計画を完了まで進めて"
```

承認済みの計画に並列実行が必要なら、`$ralph` の代わりに `$team 3:executor "承認済みの計画を並列実行して"` を使ってください。
