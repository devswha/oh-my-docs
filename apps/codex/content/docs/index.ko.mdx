**OpenAI Codex CLI를 위한 멀티 에이전트 오케스트레이션. 학습 곡선 제로.**

_Codex 프롬프트를 배울 필요 없습니다. 그냥 OMX를 사용하세요._

Oh My CodeX(OMX)는 OpenAI Codex CLI 위에서 동작하는 워크플로우 레이어입니다. Codex를 코딩 엔진으로 유지하면서, 모호한 작업을 요구사항 정리, 승인된 계획, 구현, 검증까지 이어지는 문서형 흐름으로 정리해 줍니다.

## 빠른 시작

<Steps>
  <Step>
    ### Codex CLI 설치

    ```bash
    npm i -g @openai/codex
    codex login
    ```
  </Step>
  <Step>
    ### OMX 설치 및 설정

    ```bash
    npm i -g oh-my-codex
    omx setup
    ```
  </Step>
  <Step>
    ### 설치 검증

    ```bash
    omx doctor
    codex login status
    omx exec --skip-git-repo-check -C . "Reply with exactly OMX-EXEC-OK"
    ```
  </Step>
  <Step>
    ### 첫 작업 보내기

    ```bash
    omx --madmax --high
    ```

    세션 안에서 기본 guided flow를 실행합니다.

    ```text
    $deep-interview "clarify the authentication change"
    $ralplan "approve the safest implementation path"
    $ultragoal "turn the approved plan into durable goals and complete them"
    ```
  </Step>
</Steps>
