**OpenAI Codex CLI를 위한 멀티 에이전트 오케스트레이션. 학습 곡선 제로.**

_Codex 프롬프트를 배울 필요 없습니다. 그냥 OMX를 사용하세요._

Oh My CodeX (OMX)는 OpenAI Codex CLI 위에서 동작하는 워크플로우 레이어입니다. 한 번의 호출로 특화된 에이전트 프롬프트를 조율하여 복잡한 개발 작업을 처음부터 끝까지 자동화합니다.

## 빠른 시작

### 1단계 — Codex CLI 설치

```bash
npm i -g @openai/codex
```

### 2단계 — OMX 설치

```bash
npm i -g oh-my-codex
```

### 3단계 — 설정

```bash
omx setup
```

### 4단계 — 첫 세션 스모크 테스트 실행

```bash
omx doctor
codex login status
omx exec --skip-git-repo-check -C . "Reply with exactly OMX-EXEC-OK"
```

### 5단계 — 추천 방식으로 OMX 실행

```bash
omx --madmax --high
```

그다음 세션 안에서 기본 워크플로를 시도해 보세요:

```
$deep-interview "인증 변경 사항을 명확히 해줘"
$ralplan "가장 안전한 구현 경로를 승인해줘"
$ralph "승인된 계획을 끝까지 완료해줘"
```

승인된 계획에 조율된 병렬 실행이 필요하다면 `$ralph` 대신 `$team 3:executor "승인된 계획을 병렬로 실행해줘"`를 사용하세요.
