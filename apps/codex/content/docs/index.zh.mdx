**面向 OpenAI Codex CLI 的多智能体编排。零学习曲线。**

_无需学习 Codex 提示词。直接使用 OMX 即可。_

Oh My CodeX (OMX) 是运行在 OpenAI Codex CLI 之上的工作流层。只需一次调用，它便能协调多个专用智能体提示词，端到端地自动化复杂的开发任务。

## 快速开始

### 第 1 步 — 安装 Codex CLI

```bash
npm i -g @openai/codex
```

### 第 2 步 — 安装 OMX

```bash
npm i -g oh-my-codex
```

### 第 3 步 — 初始化设置

```bash
omx setup
```

### 第 4 步 — 运行首次会话冒烟测试

```bash
omx doctor
codex login status
omx exec --skip-git-repo-check -C . "Reply with exactly OMX-EXEC-OK"
```

### 第 5 步 — 按推荐方式启动 OMX

```bash
omx --madmax --high
```

然后在该会话中尝试标准工作流：

```
$deep-interview "澄清认证改动的需求"
$ralplan "批准最安全的实现路径"
$ralph "把已批准的计划推进到完成"
```

如果已批准的计划需要并行执行，请把最后一步改为 `$team 3:executor "并行执行已批准的计划"`，而不是 `$ralph`。
