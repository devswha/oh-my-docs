**OpenAI Codex CLI 的多智能体编排。零学习曲线。**

_不必学习复杂的 Codex 提示词，直接使用 OMX。_

Oh My CodeX (OMX) 是运行在 OpenAI Codex CLI 之上的工作流层。它保留 Codex 作为编码引擎，并为模糊任务提供一条文档化路径：澄清需求、批准计划、执行实现、完成验证。

## 快速开始

<Steps>
  <Step>
    ### 安装 Codex CLI

    ```bash
    npm i -g @openai/codex
    codex login
    ```
  </Step>
  <Step>
    ### 安装并设置 OMX

    ```bash
    npm i -g oh-my-codex
    omx setup
    ```
  </Step>
  <Step>
    ### 验证安装

    ```bash
    omx doctor
    codex login status
    omx exec --skip-git-repo-check -C . "Reply with exactly OMX-EXEC-OK"
    ```
  </Step>
  <Step>
    ### 发送第一个任务

    ```bash
    omx --madmax --high
    ```

    然后在会话中运行默认 guided flow。

    ```text
    $deep-interview "clarify the authentication change"
    $ralplan "approve the safest implementation path"
    $ultragoal "turn the approved plan into durable goals and complete them"
    ```
  </Step>
</Steps>
