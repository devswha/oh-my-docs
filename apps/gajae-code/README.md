# gajae-code-docs

Official documentation site for [Gajae-Code](https://github.com/Yeachan-Heo/gajae-code), a red-claw coding-agent harness (`gjc`).

- Framework: Next.js 16 + Fumadocs 16
- Languages: English, 한국어, 中文, 日本語
- Source project: https://github.com/Yeachan-Heo/gajae-code

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run lint
npm run build
```
