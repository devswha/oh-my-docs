Gajae-Code 에이전트는 집중된 도구 표면을 통해 동작합니다. 아래는 계열별 정리이며, 이름은 업스트림 `docs/tools/` 레퍼런스와 일치합니다.

## 파일

| 도구       | 하는 일                              |
| ---------- | ------------------------------------ |
| `read`     | 파일 내용 읽기.                      |
| `write`    | 파일 생성·덮어쓰기.                  |
| `edit`     | 범위 지정 편집 (여러 편집 variant).  |
| `ast-edit` | 구조적, AST 인식 편집.               |

## 검색·탐색

| 도구              | 하는 일                          |
| ----------------- | -------------------------------- |
| `find`            | 이름/경로로 파일 찾기.           |
| `search`          | 파일 내용 검색.                  |
| `search_tool_bm25`| 랭킹 기반 내용 검색.             |
| `ast-grep`        | AST 패턴으로 구조적 코드 검색.   |
| `lsp`             | 언어 서버 진단·탐색.             |
| `resolve`         | 심볼/도구 런타임 해석.           |

## 런타임·실행

| 도구          | 하는 일                              |
| ------------- | ------------------------------------ |
| `bash`        | 셸 명령 실행 (가능 시 대화형 PTY).   |
| `eval`        | 코드 평가 (JavaScript·Python 백엔드).|
| `python-repl` | 지속 Python REPL / 커널.             |
| `debug`       | 디버깅 지원.                         |
| `ssh`         | SSH로 명령 실행.                     |
| `job`         | 백그라운드 작업 관리.                |

## 웹·외부

| 도구         | 하는 일                          |
| ------------ | -------------------------------- |
| `web_search` | 웹 검색 (여러 검색 provider).    |
| `browser`    | 실제 브라우저 구동.              |
| `github`     | GitHub 연동.                     |

## 세션·메모리

| 도구         | 하는 일                          |
| ------------ | -------------------------------- |
| `task`       | 서브에이전트 생성·조율.          |
| `todo_write` | 할 일 목록 추적·갱신.            |
| `checkpoint` | 세션 상태 스냅샷.                |
| `rewind`     | 이전 지점으로 되돌리기.          |
| `recall`     | 저장된 메모리 조회.              |
| `retain`     | 이후 조회를 위해 메모리 보존.    |
| `reflect`    | 반성적 추론 단계.                |

## 미디어·기타

| 도구             | 하는 일                  |
| ---------------- | ------------------------ |
| `render_mermaid` | Mermaid 다이어그램 렌더. |
| `inspect_image`  | 이미지 검사.             |
| `calc`           | 계산기.                  |
| `recipe`         | 재사용 작업 레시피.      |
| `ask`            | 사용자에게 질문.         |

일부 도구는 설정이나 자격증명에 의존합니다(예: `web_search` provider, `browser`는 각자 설정 필요). 이를 켜는 환경변수는 [레퍼런스](/ko/docs/reference)를 보세요.
