Gajae-Code 智能体通过精简的工具表面行动。以下按家族归类，名称与上游 `docs/tools/` 参考一致。

## 文件

| 工具       | 作用                              |
| ---------- | --------------------------------- |
| `read`     | 读取文件内容。                    |
| `write`    | 创建或覆盖文件。                  |
| `edit`     | 应用有范围的编辑（多种 variant）。|
| `ast-edit` | 结构化、AST 感知的编辑。          |

## 搜索与导航

| 工具              | 作用                       |
| ----------------- | -------------------------- |
| `find`            | 按名称/路径定位文件。      |
| `search`          | 搜索文件内容。             |
| `search_tool_bm25`| 排序的内容搜索。           |
| `ast-grep`        | 按 AST 模式做结构化搜索。  |
| `lsp`             | 语言服务器诊断与导航。     |
| `resolve`         | 符号/工具运行时解析。      |

## 运行时与执行

| 工具          | 作用                                 |
| ------------- | ------------------------------------ |
| `bash`        | 运行 shell 命令（可用时交互 PTY）。  |
| `eval`        | 评估代码（JavaScript/Python 后端）。 |
| `python-repl` | 持久 Python REPL / 内核。            |
| `debug`       | 调试支持。                           |
| `ssh`         | 通过 SSH 运行命令。                  |
| `job`         | 管理后台作业。                       |

## 网络与外部

| 工具         | 作用                       |
| ------------ | -------------------------- |
| `web_search` | 网络搜索（多个 provider）。|
| `browser`    | 驱动真实浏览器。           |
| `github`     | 与 GitHub 交互。           |

## 会话与记忆

| 工具         | 作用                       |
| ------------ | -------------------------- |
| `task`       | 生成并协调子智能体。       |
| `todo_write` | 跟踪并更新待办列表。       |
| `checkpoint` | 快照会话状态。             |
| `rewind`     | 回滚到更早的点。           |
| `recall`     | 检索已存记忆。             |
| `retain`     | 持久化记忆以备后用。       |
| `reflect`    | 反思式推理步骤。           |

## 媒体与其他

| 工具             | 作用                   |
| ---------------- | ---------------------- |
| `render_mermaid` | 渲染 Mermaid 图。      |
| `inspect_image`  | 检查图像。             |
| `calc`           | 计算器。               |
| `recipe`         | 可复用任务配方。       |
| `ask`            | 向用户提问。           |

部分工具依赖配置或凭据（例如 `web_search` provider 和 `browser` 需各自设置）。启用它们的环境变量见[参考](/zh/docs/reference)。
