description: Gajae-Code エージェントが使うツール表面 — ファイル、検索、ランタイム、Web、セッション、メディア — をファミリー別に整理。
---

Gajae-Code エージェントは絞り込まれたツール表面を通じて動作します。以下はファミリー別の整理で、名称は上流 `docs/tools/` リファレンスと一致します。

## ファイル

| ツール     | 役割                                  |
| ---------- | ------------------------------------- |
| `read`     | ファイル内容の読み取り。              |
| `write`    | ファイルの作成・上書き。              |
| `edit`     | 範囲指定の編集（複数の variant）。    |
| `ast-edit` | 構造的・AST 認識の編集。              |

## 検索とナビゲーション

| ツール            | 役割                          |
| ----------------- | ----------------------------- |
| `find`            | 名前/パスでファイルを特定。   |
| `search`          | ファイル内容を検索。          |
| `search_tool_bm25`| ランキング付き内容検索。      |
| `ast-grep`        | AST パターンで構造的検索。    |
| `lsp`             | 言語サーバーの診断・ナビ。    |
| `resolve`         | シンボル/ツールランタイム解決。|

## ランタイムと実行

| ツール        | 役割                                   |
| ------------- | -------------------------------------- |
| `bash`        | シェルコマンド実行（可能なら対話 PTY）。|
| `eval`        | コード評価（JavaScript/Python バックエンド）。|
| `python-repl` | 永続 Python REPL / カーネル。          |
| `debug`       | デバッグ支援。                         |
| `ssh`         | SSH 経由でコマンド実行。               |
| `job`         | バックグラウンドジョブ管理。           |

## Web と外部

| ツール       | 役割                           |
| ------------ | ------------------------------ |
| `web_search` | Web 検索（複数の provider）。  |
| `browser`    | 実ブラウザの操作。             |
| `github`     | GitHub との連携。              |

## セッションとメモリ

| ツール       | 役割                           |
| ------------ | ------------------------------ |
| `task`       | サブエージェントの生成・調整。 |
| `todo_write` | TODO リストの追跡・更新。      |
| `checkpoint` | セッション状態のスナップショット。|
| `rewind`     | 以前の地点へロールバック。     |
| `recall`     | 保存されたメモリの取得。       |
| `retain`     | 後で取得するためメモリを保持。 |
| `reflect`    | 内省的な推論ステップ。         |

## メディアとその他

| ツール           | 役割                     |
| ---------------- | ------------------------ |
| `render_mermaid` | Mermaid 図のレンダリング。|
| `inspect_image`  | 画像の検査。             |
| `calc`           | 電卓。                   |
| `recipe`         | 再利用可能なタスクレシピ。|
| `ask`            | ユーザーへの質問。       |

一部のツールは設定や資格情報に依存します（例：`web_search` provider と `browser` はそれぞれの設定が必要）。それらを有効化する環境変数は[リファレンス](/ja/docs/reference)を参照してください。
