## Install and Environment

    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">Gajae-Code 是 <code>gjc</code> coding-agent harness：用 session、tmux、model routing、tools 与 evidence 支撑一个较小的公开 workflow surface。</p>

  <details className="group px-4 py-3">

    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">使用 <code>bun install -g gajae-code</code>。该 package 会安装 <code>gjc</code> binary。也可使用 scoped package <code>@gajae-code/coding-agent</code>。</p>
  </details>
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>必须用 tmux 吗？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">裸 <code>gjc</code> 不强制需要，但推荐路径是 <code>gjc --tmux</code>。<code>team</code> command 需要 tmux session。</p>
  </details>
</section>

## First Use

<section className="not-prose my-6 divide-y divide-fd-border rounded-lg border border-fd-border bg-fd-card">
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>先学哪条 flow？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">先学 loop：用 <code>deep-interview</code> 澄清，用 <code>ralplan</code> 形成已评审计划，再用 <code>ultragoal</code> 推进 durable execution 与 evidence。</p>
  </details>
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>什么时候用 rlm？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">目标是 exploratory research 而不是 code mutation 时，用 <code>gjc rlm</code>。notebook 与 report artifact 会写入 <code>.gjc/_session-&lt;gjcSessionId&gt;/rlm/&lt;rlmSessionId&gt;/</code>。</p>
  </details>
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>team 是必须的吗？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">不是。<code>team</code> 只在 parallel tmux workers 确实有帮助时使用。单线程工作留在普通 loop 即可。</p>
  </details>
</section>

## Running and Limits

<section className="not-prose my-6 divide-y divide-fd-border rounded-lg border border-fd-border bg-fd-card">
  <details className="group px-4 py-3">

  </details>
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>computer-use 需要什么？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">实验性 computer tool 在受支持的 Apple Silicon macOS host 上默认可用，由 supervisor gated。可用 <code>computer.alwaysOn=false</code> 禁用。</p>
  </details>
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>command 详情在哪里？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>

  </details>
</section>
