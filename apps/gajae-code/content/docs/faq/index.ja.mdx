  </details>

## First Use

<section className="not-prose my-6 divide-y divide-fd-border rounded-lg border border-fd-border bg-fd-card">
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>最初に覚える flow は？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">まず loop を覚えてください。<code>deep-interview</code> で明確化し、<code>ralplan</code> でレビュー済み plan を作り、<code>ultragoal</code> で durable execution と evidence まで進めます。</p>
  </details>
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>rlm はいつ使いますか？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">code mutation ではなく exploratory research が目的なら <code>gjc rlm</code> を使います。notebook と report artifact は <code>.gjc/_session-&lt;gjcSessionId&gt;/rlm/&lt;rlmSessionId&gt;/</code> 配下に書かれます。</p>
  </details>
  <details className="group px-4 py-3">
    <summary className="flex cursor-pointer list-none items-center justify-between gap-4 font-medium text-fd-foreground [&::-webkit-details-marker]:hidden"><span>team は必須ですか？</span><span className="text-fd-muted-foreground transition-transform group-open:rotate-45">+</span></summary>
    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">いいえ。<code>team</code> は parallel tmux workers が本当に役立つときだけ使う任意機能です。single-threaded work は通常 loop のままで構いません。</p>
  </details>
</section>

## Running and Limits
