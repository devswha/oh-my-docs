  </details>

    <p className="mt-3 text-sm leading-7 text-fd-muted-foreground">아닙니다. <code>team</code>은 병렬 tmux worker가 실제로 도움이 될 때만 쓰는 선택지입니다. 단일 흐름 작업은 일반 루프에 두면 됩니다.</p>
