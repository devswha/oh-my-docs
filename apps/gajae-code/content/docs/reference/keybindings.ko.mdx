Gajae-Code(`gjc`)는 인터랙티브 키 바인딩 세트를 제공합니다. 정확한 코드(chord)는 빌드, 불러온 리맵, 확장이 추가한 바인딩에 따라 달라집니다.

## 활성 바인딩 보기

`gjc` 세션 안에서 `/hotkeys`를 실행하면 현재 빌드의 활성 코드를 볼 수 있어요. 이 목록은 디스크에서 불러온 리맵과 확장이 기여한 바인딩을 모두 반영하므로, 내 환경에서는 언제나 가장 정확한 뷰입니다.

## 키 바인딩 커스터마이즈

사용자 리맵은 다음 위치에 둡니다.

```text
~/.gjc/agent/keybindings.json
```

이 파일은 JSON 객체이며, 키는 키 바인딩 **액션 ID**, 값은 하나의 코드 문자열 또는 코드 문자열 배열입니다. `~/.gjc/agent/config.yml`에서는 읽지 **않으며**, 중첩된 `keybindings` 객체도 없습니다.

```json
{
  "app.model.cycleForward": "Ctrl+P",
  "app.model.selectTemporary": "Alt+P",
  "app.plan.toggle": "Alt+Shift+P"
}
```

코드 이름은 대소문자를 구분하지 않으며, UI에 표시되는 것과 같은 표기(`Ctrl+P`, `Alt+Shift+P`, `Shift+Enter`, `Ctrl+Backspace` 등)를 사용합니다.

액션을 빈 배열로 설정하면 **비활성화**됩니다.

```json
{
  "app.stt.toggle": []
}
```

오래된 비정규 액션 이름은 `keybindings.json`을 불러올 때 마이그레이션되지만, 새 설정에서는 아래의 네임스페이스 액션 ID를 사용하세요.

## 흔한 액션 ID

| 액션 ID | 기본값 | 의미 |
| --- | --- | --- |
| `app.model.cycleForward` | `Ctrl+P` | 역할 모델 앞으로 순환 |
| `app.model.cycleBackward` | `Shift+Ctrl+P` | 역할 모델 뒤로 순환 |
| `app.model.selectTemporary` | `Alt+P` | 이 세션 한정으로 모델 임시 선택 |
| `app.model.select` | `Ctrl+L` | 모델 선택기 열고 역할 설정 |
| `app.plan.toggle` | `Alt+Shift+P` | plan 모드 토글 |
| `app.history.search` | `Ctrl+R` | 프롬프트 히스토리 검색 |
| `app.tools.expand` | `Ctrl+O` | 도구 출력 펼침 토글 |
| `app.thinking.toggle` | `Ctrl+T` | thinking 블록 표시 토글 |
| `app.thinking.cycle` | `Shift+Tab` | thinking 레벨 순환 |
| `app.editor.external` | `Ctrl+G` | `$VISUAL` / `$EDITOR`에서 초안 편집 |
| `app.message.followUp` | `Ctrl+Enter` | 후속 메시지 대기열에 추가 |
| `app.message.dequeue` | `Alt+Up` | 대기 중 메시지를 에디터로 되돌리기 |
| `app.clipboard.copyLine` | `Alt+Shift+L` | 현재 줄 복사 |
| `app.clipboard.copyPrompt` | `Alt+Shift+C` | 전체 프롬프트 복사 |
| `app.stt.toggle` | `Alt+H` | 음성-텍스트 변환 녹음 토글 |

이는 흔한 바인딩입니다. 확장이 기여한 액션을 포함한 빌드 정확 전체 목록은 `/hotkeys`로 확인하세요.
